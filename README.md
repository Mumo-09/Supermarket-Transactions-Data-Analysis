## Supermarket Transactions Data Analysis

This project analyzes a supermarket transactions dataset using Microsoft Excel to answer business questions about customer purchases and spending patterns.

### Project Overview

The analysis was completed as part of a job simulation exercise. Using Excel formulas, transactional data was analyzed to extract meaningful insights from a supermarket sales dataset.

### Dataset

- **File:** `supermarket_transactions.csv`
- **Duration:** 3 years of supermarket transaction data
- **Tool Used:** Microsoft Excel

### Objectives

The analysis answers the following business questions:

1. How many apples were purchased using cash across all store locations?
2. What was the total cash spent on those apple purchases?
3. How much money was spent by non-member customers at the Bakershire store across all payment methods?

### Excel Functions Used

- `SUMIFS()`
- Data Filtering
- Spreadsheet Analysis
- Conditional Aggregation

### Deliverables

- An analyzed Excel workbook
- An **Analysis** worksheet containing:
  - Business questions
  - Excel formulas used
  - Calculated results

### Skills Demonstrated

- Data Cleaning
- Data Analysis
- Microsoft Excel
- Spreadsheet Functions
- Business Intelligence
- Data Interpretation

### Outcome

Successfully analyzed supermarket transaction data and generated insights using Excel's conditional aggregation functions to support business decision-making.
